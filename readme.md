# Pixiv爬虫

1：排行榜模式
-------------
    1：综合排行榜
         模式
             日榜为输入日期
             周榜为输入日期向前推一周，比如输入20200807则为20200801~20200807
             月榜同理，向前推一个月
             新人排行
             男性排行
             女性排行
    2：插画排行榜
         模式
             日榜为输入的日期
             周榜为输入的日期向前推一周，比如输入20200807则为20200801~20200807
             月榜同理，向前推一个月
             新人排行
            
    PS：
    1：所有模式均有单ID页面最大上限下载设定，请自行输入单页最大插画数量
    2：所有模式都可以选定下载前多少名
    
2:画师模式
   ------
    1:输入喜欢的画师的ID 下载该画师所有插画
